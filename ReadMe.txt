Task:
	Задан следующий интерфейс

import java.rmi.Remote;

public interface DateTime extends Remote
{
    String getDate () throws RemoteException;
    String getTime () throws RemoteException;
}

	Напишите приложение, в котором:
Серверная часть приложения реализует удаленную службу, определенную интерфейсом DateTime. 
Метод getDate () должен возвращать текущую дату в виде строки, например, «29.11.2012». 
Метод getTime() должен возвращать текущее время в виде строки, например, «11:46»;
Клиентская часть приложения вызывает метод удаленной службы getDate (), 
заданный интерфейсом DateTime, и выводит полученные результаты;
Кроме этого, клиентская часть приложения должна по специальной команде или по таймеру вызывать метод getTime () 
удалённой службы и отображать полученное время.
Клиентская часть приложения должна по отдельной команде завершать приложение.
